# 🎯 Jogo do Número Secreto

Aplicação simples onde o usuário tenta adivinhar um número aleatório.

## 📌 Funcionalidades

- Geração de número aleatório
- Tentativas do usuário
- Feedback (maior ou menor)

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript

## ▶️ Como executar

Abra o arquivo `index.html` no navegador.

## 📚 Objetivo

Praticar lógica de programação, estruturas condicionais e interação com o usuário.
